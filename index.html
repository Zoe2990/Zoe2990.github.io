Intelligent Cybersecurity Q&A System

Work Content:

  • Designed an intelligent chatbot that understands user questions and generates safe SQL to retrieve asset, usage, and risk data from the backend — no manual query writing required.
  • SQL generation, internal file retrieval, and online search to generate comprehensive answers from both structured databases and unstructured content.
  • Built a multi-step LangGraph agent flow to manage conversation state, fallback handling, and validation logic for robust performance.
  • Integrated contextual prompting and multi-source RAG pipelines to resolve ambiguity and improve accuracy.
  • Enabled multi-tenant data isolation based on organization ID to ensure security and privacy in enterprise environments.

Impact:

  • Empowered non-technical users to query cybersecurity data without writing SQL or checking documentation.
  • Reduced reliance on technical staff for data access.
  • Created a modular architecture ready to scale into a full-featured enterprise data assistant.

Tech Stack:

LangGraph, SQLAlchemy, AWS Bedrock, AWS Lambda, AWS EC2, PostgreSQL, pgvector, OpenSearch, Tavily, FAISS, Retrieval-Augmented Generation (RAG), Prompt Engineering, Agent Design & Tool Calling.

ThreatMap: AI-Powered Threat Modeling Agent


Work Content:

  • Built a Retrieval-Augmented Generation (RAG) pipeline to convert user inputs—blogs, CVEs, or natural language descriptions—into structured threat modeling results.
  • Mapped each step of the threat model (Tactic → Technique → Sub-technique) to the MITRE ATT&CK framework for clarity and standardization.
  • Delivered interpretable, report-ready threat models with structured outputs for security analysis.


Impact:

  • Enabled “one sentence in, full threat model out” — streamlining what used to be a manual and expert-heavy task.
  • Made threat modeling faster, explainable, and accessible even for teams without deep cybersecurity expertise.

Tech Stack:

LangChain, FAISS, OpenAI GPT, GPT-4o, MITRE ATT&CK, Custom Web Crawler, Vector Embedding, Retrieval-Augmented Generation Pipelines
